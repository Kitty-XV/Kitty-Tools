编码说明

组件编码 (encodeURIComponent)

将字符串编码为 URI 组件，会编码所有特殊字符，包括 /?:@&=+$,# 等。适用于编码 URL 参数。

URI编码 (encodeURI)

将字符串编码为完整的 URI，不会编码 /?:@&=+$,# 等特殊字符。适用于编码完整 URL。

示例：

原文：https://example.com/?name=测试&id=123
组件编码：https%3A%2F%2Fexample.com%2F%3Fname%3D%E6%B5%8B%E8%AF%95%26id%3D123
URI编码：https://example.com/?name=%E6%B5%8B%E8%AF%95&id=123
